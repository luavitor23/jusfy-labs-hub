# Prompt inicial para uma nova conversa

Use este texto quando quiser reforçar a retomada do projeto:

> Estamos trabalhando no projeto **Construtor de Criativos**. Leia primeiro o arquivo `AGENTS.md` da raiz do projeto e acesse a página **Construtor de Criativos** no Notion. Use o Notion como fonte de verdade, leia o plano de ação e as regras de marca, confirme pelo MCP a estrutura, os IDs das pastas e os mestres do Canva e somente depois execute esta solicitação: **[descrever o novo criativo]**.

## Quando abrir uma nova conversa

- Abra uma nova conversa para uma nova campanha, novo template ou novo lote independente.
- Continue na mesma conversa durante criação, ajustes, prévias e aprovação de uma mesma entrega.
- Não mude de conversa enquanto existir uma edição pendente ou não publicada no Canva.
