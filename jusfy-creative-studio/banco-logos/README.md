# Banco de logos do Studio

O catálogo usa o arquivo `LOGOS.svg` fornecido pelo time e mantém as marcas regionais separadas da Jusfy.

- A regional selecionada define a logo e substitui automaticamente `OAB/UF` nas copies do Notion.
- As caixas de seleção permitem gerar versões em massa para várias regionais; com copies marcadas, o lote é a combinação de copies × logos.
- Os recortes da prancha `LOGOS.svg` removem automaticamente o fundo branco ou quase branco para acompanhar o fundo da peça.
- A logo Jusfy permanece na versão oficial.
- Novas marcas devem ser adicionadas ao `catalog.json` somente após revisão visual.
